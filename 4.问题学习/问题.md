# 1. Vector3是什么，有什么作用（代码：3.dome/index.js）？
Vector3是three.js中的一个向量类，用于表示三维空间中的向量。它可以用于表示位置、方向、速度等。
# 2. Vector3的常用方法有哪些（代码：3.dome/index.js）？
Vector3的常用方法有：
- set(x, y, z)：设置向量的xyz分量
- add(v)：向量加法
- sub(v)：向量减法 （代码：3.dome/index.js）
- mulScalar(s)：向量与标量乘法
- normalize()：向量归一化
- dot(v)：向量点积，用于计算两个向量的相似度，范围[-1, 1]，越接近1表示越相似 （代码：3.dome/index.js）
- cross(v)：向量叉积，用于计算两个向量的正交向量，方向垂直于这两个向量所在的平面
# 3. Object3D是什么，有什么作用（代码：3.dome/index.js）？
Object3D 是 Three.js 场景中所有可放进 scene 的东西的父类，用来管理「位置、旋转、缩放、层级关系」。
